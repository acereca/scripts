# Scripts
Script collection, from a to z.

Contained here are mostly independent small scripts.

## Install

```zsh
./install.zsh
```

## Usage

run any script contained inside this folder with

```zsh
ssc <script-name>
```

where `<script-name>` is the files base-name (w/o extension)
e.g.:

```zsh
ssc select.music
```
